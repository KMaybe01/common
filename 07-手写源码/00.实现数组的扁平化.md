# 实现数组的扁平化

将多维嵌套数组扁平化为一维数组，提供多种实现方式：栈、递归+reduce、扩展运算符、toString 以及 ES6 flat。

## 流程图

```mermaid
    C2 -->|是| C3[递归调用 flatten1]
    C2 -->|否| C4[concat 到结果]

    E --> E1[toString 转字符串]
    E --> E2[split 转数组]
    E --> E3[仅适用于纯字符串/数字]

```

## 原始代码

```javascript
let arr = [1, [2, [3, 4, 5]]];


// 栈
function flattenDeep(arr) {
  const result = [];
  // 将数组元素拷贝至栈，直接赋值会改变原数组
  const stack = [...arr];
  // 如果栈不为空，则循环遍历
  while (stack.length !== 0) {
    const val = stack.pop();
    if (Array.isArray(val)) {
      // 如果是数组再次入栈，并且展开了一层
      stack.push(...val);
    } else {
      // 如果不是数组，就用头插法插入到结果数组中
      result.unshift(val);
    }
  }
  return result;
}

// 测试
var test = ["a", ["b", "c"], ["d", ["e", ["f"]], "g"]];
console.log(flattenDeep(test));  // [ 'a', 'b', 'c','d', 'e', 'f', 'g']


// console.log(flatten1(arr)); // [1, 2, 3, 4，5]
//方法一: 递归 + reduce() + concat()

function flatten1(array) {
  return array.reduce((pre, item) => {
    if (Array.isArray(item) && item.some((cItem) => Array.isArray(cItem))) {
      return pre.concat(flatten1(item));
    } else {
      return pre.concat(item);
    }
  }, []);
}

//递归实现
function flatten1(arr) {
  let result = [];
  for (let i = 0; i < arr.length; i++) {
    if (Array.isArray(arr[i])) {
      result = result.concat(flatten1(arr[i]));
    } else {
      result.push(arr[i]);
    }
  }
  return result;
}

//扩展运算符实现
function flatten3(arr) {
  while (arr.some((item) => Array.isArray(item))) {
    arr = [].concat(...arr);
  }
  return arr;
}

//split 和 toString
function flatten4(arr) {
  return arr.toString().split(",");
}


//ES6 中的 flat
function flatten5(arr) {
  return arr.flat(Infinity);
}
```

## 逐段解析

### 方式一：栈
- 先将原数组拷贝到栈（`[...arr]`）
- while 循环从栈中 pop 元素，如果是数组则展开后 push 回栈
- 如果不是数组则用 `unshift` 插入结果数组头部（因为栈是 LIFO）
- **优点**：不递归，无调用栈溢出风险

### 方式二：递归 + reduce + concat
- `reduce` 遍历数组每个元素
- 如果元素是数组且内部还有嵌套数组，递归调用 `flatten1`
- 使用 `concat` 将结果合并到累加器
- **简化递归版**：直接用 for 循环遍历，遇到数组就递归 concat

### 方式三：扩展运算符 + some + concat
- `some` 检测数组是否还有嵌套数组
- `[].concat(...arr)` 每次展开最外层一层嵌套
- while 循环直到全部展开
- **巧妙**：扩展运算符只能展开一层，配合循环实现完全扁平化

### 方式四：toString + split
- `arr.toString()` 将嵌套数组转为逗号分隔的字符串
- `split(",")` 再转回数组
- **局限**：所有元素都变成字符串，元素类型丢失

### 方式五：ES6 flat
- `arr.flat(Infinity)` 原生方法，指定深度为无穷大
- **最简洁**，但需要考虑浏览器兼容性

## 复杂度分析
- **栈方式**：时间复杂度 O(n)，空间复杂度 O(n)
- **递归方式**：时间复杂度 O(n)，空间复杂度 O(n)（递归调用栈）
- **扩展运算符**：时间复杂度 O(n²)，每次展开需要遍历
- **推荐**：栈方式性能最优，ES6 flat 写法最简洁
