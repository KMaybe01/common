# 实现柯里化函数

## 简介

柯里化（Currying）是将一个接收多个参数的函数转换为一系列接收单个参数的函数的过程。其核心思想是：**当参数未收集够时，继续返回函数收集参数；当参数收集完毕时，一次性执行原函数**。本文展示了多种柯里化实现方式。

## 执行流程

```mermaid
flowchart TD
    A[调用 currying(fn)] --> B[返回 temp 函数]
    B --> C[temp(newArgs)]
    C --> D{newArgs.length > 0?}
    D -->|是| E[合并 args 和 newArgs]
    E --> F[返回 temp 函数自身]
    D -->|否| G[执行 fn(...args)]
    G --> H[清空 args]
    H --> I[返回执行结果]
    
    J[curry(add, ...args)] --> K{fn.length <= args.length?}
    K -->|是| L[立即执行 fn]
    K -->|否| M[返回 curry.bind(null, fn, ...args)]
```

## 代码实现

```javascript
//粗暴版
function add2(a) {
  return function (b) {
    return function (c) {
      return a + b + c;
    };
  };
}
console.log(add2(1)(2)(3)); //6

//完美法
function add(...args) {
  return args.reduce((a, b) => a + b);
}

function currying(fn) {
  let args = [];
  return function temp(...newArgs) {
    if (newArgs.length) {
      args = [...args, ...newArgs];
      return temp;
    } else {
      let val = fn.apply(this, args);
      args = [];
      return val;
    }
  };
}
let addCurry = currying(add);
console.log(add(1, 2, 3, 4, 5)); //15
console.log(addCurry(1)(2)(3)(4, 5)()); //15
console.log(addCurry(1)(2)(3, 4, 5)()); //15
console.log(addCurry(1)(2, 3, 4, 5)()); //15

//箭头函数
const curry = (fn) => {
  let argList = [];
  return function curryFn(...args) {
    if (!args.length) {
      return fn(...argList);
    } else {
      argList = argList.concat(args);
      return curryFn;
    }
  };
};
const addFn = (...args) => args.reduce((total, cur) => total + cur, 0);
console.log(addFn(1, 2, 3, 4)); //10
const add3 = curry(addFn);
console.log(add3(1, 2)(3)(4)()); //10

// ES6 实现
function curry(fn, ...args) {
  return fn.length <= args.length ? fn(...args) : curry.bind(null, fn, ...args);
}

let addCurry2 = curry(add, 1, 5, 8);
console.log(addCurry2); //=>14
```

## 逐行解析

### 粗暴版
- 硬编码返回多层嵌套函数，每一层接收一个参数。不通用，只适用于固定参数个数。

### 完美版（闭包收集参数）
- **`let args = []`**：通过闭包维护一个参数收集数组。
- **`temp(...newArgs)`**：返回的代理函数，每次调用时判断参数情况。
- **`if (newArgs.length)`**：如果有传入参数，则合并到 `args` 中并返回 `temp` 自身继续等待参数。
- **`fn.apply(this, args)`**：当无参调用时（`addCurry(1)(2)(3)()`），执行原函数并传入所有收集的参数。
- **`args = []`**：执行完后清空参数数组，保证再次调用时状态正确。

### 箭头函数版
- 逻辑与完美版相同，使用 `concat` 合并参数数组。
- 使用箭头函数和更简洁的语法。

### ES6 版（递归 + bind）
- **`fn.length <= args.length`**：判断原函数的形参个数是否小于等于已收集的参数个数。
- 如果参数已够，直接执行 `fn(...args)`。
- 如果参数不够，通过 `curry.bind(null, fn, ...args)` 返回一个绑定了已收集参数的新函数，等待接收剩余参数。

## 复杂度分析

- **时间复杂度**：O(n) — n 为最终参数总数，每次参数收集 O(1)，最终执行 O(n)
- **空间复杂度**：O(n) — 需要收集保存所有参数
