# 实现数组的 Array.prototype.filter 方法

## 简介

手动实现数组的 `filter` 方法，对数组的每个元素执行测试函数，返回所有满足条件的元素组成的新数组。

## 执行流程

```mermaid
flowchart TD
    A[开始] --> B[检查参数合法性]
    B --> C{arr是数组且\nfilterCallback是函数?}
    C -->|否| D[返回空数组 []]
    C -->|是| E[初始化空数组 result]
    E --> F[遍历数组每个元素]
    F --> G[调用 filterCallback(arr[i], i, arr)]
    G --> H{返回值为真?}
    H -->|是| I[将 arr[i] push 到 result]
    H -->|否| J[跳过该元素]
    I --> K{遍历完成?}
    J --> K
    K -->|否| F
    K -->|是| L[返回 result]
    L --> M[结束]
```

## 代码实现

```javascript
function filter(arr, filterCallback) {
    // 首先，检查传递的参数是否正确。
    if (!Array.isArray(arr) || !arr.length || typeof filterCallback !== 'function') {
        return [];
    } else {
        let result = [];
        // 每次调用此函数时，我们都会创建一个 result 数组
        // 因为我们不想改变原始数组。
        for (let i = 0, len = arr.length; i < len; i++) {
            // 检查 filterCallback 的返回值是否是真值
            if (filterCallback(arr[i], i, arr)) {
                // 如果条件为真，则将数组元素 push 到 result 中
                result.push(arr[i]);
            }
        }
        return result;
    }
}
```

## 逐行解析

1. **第 1 行**: 定义 `filter` 函数，接收数组 `arr` 和回调函数 `filterCallback`。
2. **第 3 行**: 参数校验，若 `arr` 不是数组、为空或 `filterCallback` 不是函数则返回空数组。
3. **第 6 行**: 初始化 `result` 数组存放筛选结果。
4. **第 9-14 行**: 遍历数组，对每个元素执行 `filterCallback(arr[i], i, arr)`，若返回值为真（truthy），则将当前元素推入 `result`。
5. **第 16 行**: 返回筛选后的新数组。

## 复杂度分析

- **时间复杂度**: O(n)，n 为数组长度，每个元素执行一次回调。
- **空间复杂度**: O(n)，最坏情况下所有元素都满足条件，新数组与原数组等长。
