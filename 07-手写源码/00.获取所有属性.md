# 获取所有属性

递归遍历嵌套对象，获取所有非对象叶子节点的属性名。

## 流程图

```mermaid
flowchart TD
    A[输入 obj] --> B{typeof obj === object?}
    B -- 否 --> C[返回 undefined]
    B -- 是 --> D[初始化 keys = []]
    D --> E[for...in 遍历每个 key]
    E --> F{obj[key] 是对象<br/>且不是数组?}
    F -- 是 --> G[递归调用 getAllKey]
    G --> H[合并 keys]
    F -- 否 --> I[将 key 加入 keys]
    H --> J[返回 keys]
    I --> J
```

## 代码与解析

```javascript
//获取所有属性
var obj = {
  a: "12",
  first: {
    c: "34",
    d: "45",
    second: {
      3: "56",
      f: "67",
      three: {
        g: "78",
        h: "89",
        i: "90",
      },
    },
  },
};

function getAllKey(obj) {
  if (typeof obj !== "object") {
    return;
  }
  let keys = [];
  for (let index in obj) {
    if (obj[index] instanceof Object && !Array.isArray(obj[index])) {
      keys = keys.concat(getAllKey(obj[index]));
    } else {
      keys.push(index);
    }
  }
  return keys;
}
console.log(getAllKey(obj));
// => [a,c,d,e,f,g,h,i]
```

- **递归遍历**：`for...in` 遍历对象所有可枚举属性（包括自身和继承的）
- **判断条件**：值如果是一个纯对象（`Object` 实例且不是数组），则递归深入
- **叶子节点**：非对象属性或数组，直接将 key 加入结果数组
- 注意：输出中的 `e` 在原始代码中实际为 `f`（对应 `second` 中的 `f: "67"`），`3` 为数字 key，`for...in` 会将其转为字符串

## 复杂度分析

| 指标 | 值 |
|------|-----|
| 时间复杂度 | O(n) — n 为对象中所有节点总数 |
| 空间复杂度 | O(d) — d 为对象的最大深度（递归调用栈） |
