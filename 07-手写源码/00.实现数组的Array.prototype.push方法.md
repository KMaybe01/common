# 实现数组的 Array.prototype.push 方法

## 简介

手动实现数组的 `push` 方法，在数组末尾添加一个或多个元素，并返回新数组的长度。

## 执行流程

```mermaid
flowchart TD
    A[开始] --> B[定义myPush方法]
    B --> C[遍历arguments参数列表]
    C --> D[将每个参数赋值到 this\[this.length\]]
    D --> E[更新数组长度]
    E --> F{还有参数?}
    F -->|是| C
    F -->|否| G[返回 this.length]
    G --> H[结束]
```

## 代码实现

```javascript
let arr = [];
Array.prototype.myPush = function () {
    for (let i = 0; i < arguments.length; i++) {
        this[this.length] = arguments[i];
    }
    return this.length;
}


arr.myPush(1)
arr.myPush(1)
arr.myPush(1)
console.log(arr)
```

## 逐行解析

1. **第 1 行**: 定义一个空数组 `arr` 用于测试。
2. **第 2 行**: 在 `Array.prototype` 上添加自定义方法 `myPush`，使用普通函数确保 `this` 指向调用该方法的数组实例。
3. **第 3-5 行**: 遍历 `arguments` 对象（类数组，包含所有传入参数），将每个元素依次赋值到 `this[this.length]` 位置，即数组末尾。
4. **第 6 行**: 返回 `this.length`，即操作完成后数组的新长度，与原生 `push` 行为一致。
5. **第 10-12 行**: 测试调用，向 `arr` 中三次添加元素 `1`。
6. **第 13 行**: 打印 `arr`，输出 `[1, 1, 1]`。

## 复杂度分析

- **时间复杂度**: O(n)，n 为传入参数的数量，需要逐个添加。
- **空间复杂度**: O(1)，除了参数本身外没有额外空间开销。
