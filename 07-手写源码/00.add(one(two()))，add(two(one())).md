# add(one(two())) / add(two(one()))

## 简介

实现 `add(one(two()))` 和 `add(two(one()))` 都输出 `3`。其核心思路是：`one` 和 `two` 在无参调用时返回自身的数值，在有参调用时将该参数与自己组合成数组返回；`add` 函数接收一个数组并对所有元素求和。

## 执行流程

```mermaid
flowchart TD
    A[one(two())] --> B[two 无参调用返回 2]
    B --> C[one(2) 返回 [2, 1]]
    C --> D[add([2,1]) 求和 => 3]
    
    E[two(one())] --> F[one 无参调用返回 1]
    F --> G[two(1) 返回 [1, 2]]
    G --> H[add([1,2]) 求和 => 3]
```

## 代码实现

```javascript
function add() {
    return arguments[0].reduce((a,b)=>a+b)
}

function one() {  
    if(arguments.length==0){
        return 1
    } else {
        return [arguments[0],1]
    }
}
function two() {
    if(arguments.length==0){
        return 2
    } else {
        return [arguments[0],2]
    }
}

console.log(add(one(two())));  //3
console.log(add(two(one())));  //3
```

## 逐行解析

- **`function add()`**：接收一个数组参数（位于 `arguments[0]`），调用数组的 `reduce` 方法对所有元素进行累加求和。
- **`function one()`**：判断 `arguments.length` 是否为 0。如果无参调用，直接返回数值 `1`；如果有参调用，将参数 `arguments[0]` 与自己（数字 `1`）组合成一个数组 `[arguments[0], 1]` 返回。
- **`function two()`**：逻辑与 `one` 相同，无参返回 `2`，有参返回 `[arguments[0], 2]`。
- **执行过程**：`one(two())` 先从内到外执行：`two()` 无参返回 `2`，然后 `one(2)` 返回 `[2, 1]`，最后 `add([2, 1])` 求和得 `3`。`two(one())` 同理。

## 复杂度分析

- **时间复杂度**：O(n) — reduce 求和需要遍历数组所有元素
- **空间复杂度**：O(n) — 需要创建数组存储参数
