# 生成随机字符串

从指定的字符集中随机选取字符，生成指定长度的随机字符串。

## 流程图

```mermaid
flowchart TD
    A[输入长度 len] --> B[定义字符集 chars]
    B --> C[获取字符集长度 strLen]
    C --> D[初始化 randomStr = '']
    D --> E[循环 len 次]
    E --> F[生成 0~strLen-1 随机索引]
    F --> G[chars.charAt(索引)]
    G --> H[追加到 randomStr]
    H --> E
    E --> I[返回 randomStr]
```

## 代码与解析

```javascript
const randomString = (len) => {
  let chars = "ABCDEFGHJKMNPQRSTWXYZabcdefhijkmnprstwxyz123456789";
  let strLen = chars.length;
  let randomStr = "";
  for (let i = 0; i < len; i++) {
    randomStr += chars.charAt(Math.floor(Math.random() * strLen));
  }
  return randomStr;
};
console.log(randomString(5));
```

- **字符集**：去掉了易混淆的字符（如 `0`/`O`、`1`/`l`/`I`）
- `Math.random()` 生成 `[0, 1)` 随机浮点数
- `Math.floor(Math.random() * strLen)` 生成 `[0, strLen-1]` 的整数索引
- `chars.charAt(index)` 获取对应字符
- 循环 `len` 次拼接成最终字符串

## 复杂度分析

| 指标 | 值 |
|------|-----|
| 时间复杂度 | O(n) — n 为生成字符串的长度 |
| 空间复杂度 | O(n) — 存储结果字符串 |
