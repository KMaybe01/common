# 手写 apply 函数

## 简介

`apply` 函数用于调用一个函数，并显式指定其 `this` 上下文，参数以数组形式传入。手动实现 `apply` 的核心技巧是：**将函数临时作为上下文对象的方法来调用**，调用完毕后删除该方法即可。

## 执行流程

```mermaid
flowchart TD
    A[fn.myApply(context, argsArr)] --> B{判断 this 是否为函数}
    B -->|否| C[抛出 TypeError]
    B -->|是| D[context = context || window]
    D --> E[context.fn = this]
    E --> F{arguments[1] 存在?}
    F -->|是| G[result = context.fn(...argsArr)]
    F -->|否| H[result = context.fn()]
    G --> I[delete context.fn]
    H --> I
    I --> J[返回 result]
```

## 代码实现

```javascript
Function.prototype.myApply = function (context) {
  if (typeof this !== "function") {
    throw new TypeError("Error");
  }
  let result = null;
  context = context || window;
  context.fn = this;
  if (arguments[1]) {
    result = context.fn(...arguments[1]);
  } else {
    result = context.fn();
  }
  delete context.fn;
  return result;
}

const obj = {
  value: 1,
}

function fn() {
  console.log(this.value, ...arguments); // 1 1 2 3 4 5
}

fn.myApply(obj, [1, 2, 3, 4, 5]);
```

## 逐行解析

- **`typeof this !== "function"`**：判断调用者是否为函数。
- **`context = context || window`**：如果未传入 `context`，则默认指向全局对象 `window`（浏览器环境）。
- **`context.fn = this`**：将当前函数临时作为 `context` 对象的一个属性 `fn`。
- **`if (arguments[1])`**：判断是否传入了参数数组。
- **`context.fn(...arguments[1])`**：通过 `context.fn()` 调用函数，此时函数内部的 `this` 指向 `context`。使用展开运算符将数组元素展开为参数。
- **`result = context.fn()`**：无参数时的调用方式。
- **`delete context.fn`**：调用完毕后删除临时添加的属性，保持上下文对象的干净。
- **`return result`**：返回函数执行结果。

## 复杂度分析

- **时间复杂度**：O(1) — 固定操作次数
- **空间复杂度**：O(1) — 仅使用常数额外空间
