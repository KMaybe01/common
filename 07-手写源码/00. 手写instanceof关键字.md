# 手写 instanceof 关键字

## 简介

`instanceof` 用于检测一个对象是否是一个构造函数的实例。其底层原理是：**沿着实例的 `__proto__` （隐式原型）链向上查找，直到找到等于构造函数的 `prototype`（显式原型）为止**。如果找到则返回 `true`，查找到原型链终点（`null`）则返回 `false`。

## 执行流程

```mermaid
flowchart TD
    A[instance_of(L, R)] --> B[取 O = R.prototype]
    B --> C[取 L = L.__proto__]
    C --> D{L === null?}
    D -->|是| E[返回 false]
    D -->|否| F{O === L?}
    F -->|是| G[返回 true]
    F -->|否| H[L = L.__proto__ 继续上溯]
    H --> D
```

## 代码实现

```javascript
function instance_of(L, R) {
    var O = R.prototype;
    L = L.__proto__;
    while (true) {
        if (L === null) return false;
        if (O === L){
         return true;   
        }
        L = L.__proto__;
    }
}

let arr = [];
console.log(instance_of(arr, Array)); //=>true 
console.log(instance_of(arr, RegExp)); //=>false
console.log(instance_of(arr, Object));  //=>true 

console.log(arr instanceof Array); //=>true 
console.log(arr instanceof RegExp); //=>false
console.log(arr instanceof Object); //=>true 

function Fn() {
    this.x = 100;
}
Fn.prototype = Object.create(Array.prototype);
let f = new Fn;
console.log(f, f instanceof Array); //Array { x: 100 } true

console.log(1 instanceof Number); // false
```

## 逐行解析

- **`var O = R.prototype`**：获取构造函数 R 的显式原型对象。
- **`L = L.__proto__`**：获取实例 L 的隐式原型链。也可以使用 `Object.getPrototypeOf(L)`。
- **`while (true)`**：无限循环，沿原型链不断上溯。
- **`if (L === null) return false`**：如果上溯到了 `null`（原型链的终点），说明未找到匹配，返回 `false`。
- **`if (O === L) return true`**：如果当前原型对象等于构造函数的 `prototype`，说明匹配成功，返回 `true`。
- **`L = L.__proto__`**：继续沿原型链向上查找。
- **特殊情况**：`Fn.prototype = Object.create(Array.prototype)` 将 `Fn` 的原型指向数组实例，因此 `f instanceof Array` 返回 `true`。
- **基本类型**：`1 instanceof Number` 返回 `false`，因为基本类型不是对象，没有原型链。

## 复杂度分析

- **时间复杂度**：O(n) — n 为原型链的长度
- **空间复杂度**：O(1) — 仅使用常数额外空间
