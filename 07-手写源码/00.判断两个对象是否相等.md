# 判断两个对象是否相等

## 简介

实现深度比较函数 `deepEqual`，递归比较两个对象的每个属性是否相等，支持嵌套对象。

## 执行流程

```mermaid
flowchart TD
    A[开始] --> B[比较 x === y]
    B -->|是| C[返回 true]
    B -->|否| D{x 和 y 都是对象?}

    D -->|是| F{keys 长度相等?}
    F -->|否| G[返回 false]
    F -->|是| H[遍历 x 的属性]
    H --> I{y 有该属性?}
    I -->|否| J[返回 false]
    I -->|是| K[递归比较 x[prop] 和 y[prop]]
    K --> L{相等?}
    L -->|否| J
    L -->|是| M{还有属性?}
    M -->|是| H
    M -->|否| N[返回 true]
    C --> O[结束]
    E --> O
    G --> O
    J --> O
    N --> O
```

## 代码实现

```javascript
const deepEqual = function (x, y) {
    if (x === y) {
        return true;
    } else if ((typeof x === 'object' && x != null) && (typeof y === 'object' && y != null)) {
        if (Object.keys(x).length != Object.keys(y).length) {
            return false;
        }
        for (let prop in x) {
            if (y.hasOwnProperty(prop)) {
                if (!deepEqual(x[prop], y[prop])) {
                    return false;
                }
            } else {
                return false;
            }
        }
        return true;
    }
    return false;
}

console.log(deepEqual({}, { m: 1, a: 1 })) // false
console.log(deepEqual({ m: 1, a: 1 }, { m: 1, a: 1 })) // true
```

## 逐行解析

1. **第 7 行**: 定义 `deepEqual` 函数，接收两个待比较对象 `x` 和 `y`。
2. **第 8-10 行**: 若 `x === y`（同一引用或原始值相等），直接返回 `true`。
3. **第 11-25 行**: 若两者都是对象且不为 `null`，先比较 `Object.keys` 数量，不等则返回 `false`；再遍历 `x` 的属性，检查 `y` 是否有同名属性且递归比较值是否相等。
4. **第 26 行**: 其他情况（如一方为 `null`、非对象等）返回 `false`。

## 复杂度分析

- **时间复杂度**: O(n)，n 为所有嵌套属性的总数，每个属性访问一次。
- **空间复杂度**: O(d)，d 为对象嵌套深度，递归调用栈深度。
