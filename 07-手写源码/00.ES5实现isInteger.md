# ES5 实现 isInteger

## 简介

ES5 中没有 `Number.isInteger` 方法，需要通过其他方式来判断一个值是否为整数。常见的实现方法有三种：取整比较法、异或运算法和取余法。核心思路是：先确保值是 `number` 类型且有穷，再判断其是否为整数。

## 执行流程

```mermaid
flowchart TD
    A[输入 value] --> B{typeof === 'number'?}
    B -->|否| C[返回 false]
    B -->|是| D{isFinite(value)?}
    D -->|否| C
    D -->|是| E[判断是否为整数]
    E --> F[方法1: Math.floor === value]
    E --> G[方法2: x ^ 0 === x]
    E --> H[方法3: x % 1 === 0]
    F --> I[返回结果]
    G --> I
    H --> I
```

## 代码实现

```javascript
//取整
Number.isInteger = function (value) {
    return typeof value === "number" &&
        isFinite(value) &&
        Math.floor(value) === value;
};
//异或运算
function isInteger(x) {
    return typeof value === "number" &&
        isFinite(value) && x ^ 0 === x
}

//取余
function isInteger(x) {
    return typeof value === "number" &&
        isFinite(value) && (x % 1 === 0)
}
```

## 逐行解析

### 方法一：取整比较法
- **`typeof value === "number"`**：先判断是否为 number 类型，排除字符串等类型。
- **`isFinite(value)`**：排除 `Infinity`、`-Infinity` 和 `NaN` 等非有限值。
- **`Math.floor(value) === value`**：向下取整后与原值相等，说明是整数。

### 方法二：异或运算法
- **`x ^ 0 === x`**：按位异或运算会将小数部分截断，如果异或 0 后等于自身，说明没有小数部分，即为整数。
- 注意：该写法有 BUG，变量名应为 `x` 而非 `value`。异或运算只能处理 32 位整数范围内的值。

### 方法三：取余法
- **`x % 1 === 0`**：对 1 取余结果为 0，说明没有小数部分，即为整数。
- 注意：变量名存在同样的问题，应为 `x` 而非 `value`。

## 复杂度分析

- **时间复杂度**：O(1) — 固定次数的操作
- **空间复杂度**：O(1) — 不额外占用空间
