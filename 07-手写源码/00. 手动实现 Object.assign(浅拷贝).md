# 手动实现 Object.assign（浅拷贝）

## 简介

`Object.assign(target, ...sources)` 用于将所有可枚举的自身属性从一个或多个源对象复制到目标对象，实现浅拷贝。手动实现时需要处理：过滤 null/undefined 源对象、遍历可枚举自身属性、跳过原型链上的属性。

## 执行流程

```mermaid
flowchart TD
    A[myAssign(target, ...src)] --> B[遍历每个 src 源对象]
    B --> C{src[i] 不为 null/undefined?}
    C -->|是| D[for...in 遍历源对象属性]
    D --> E{obj.hasOwnProperty(key)?}
    E -->|是| F[target[key] = src[i][key]]
    E -->|否| G[跳过原型链属性]
    C -->|否| H[跳过该源对象]
    B --> I[返回 target]
```

## 代码实现

```javascript
//能实现一维对象深拷贝
Object.myAssign = function (target, ...src) {
  for (let i = 0; i < src.length; i++) {
    if (src[i] !== null || src[i] !== undefined) {
      for (let key in src[i]) {
        if (src[i].hasOwnProperty(key)) {
          target[key] = src[i][key];
        }
      }
    }
  }
  return target;
};

const proto = {
  p: "proto",
};
const obj1 = {
  a: "aa",
};
const obj2 = {
  b: "bb",
};
const obj3 = Object.create(proto, {
  c: {
    value: "cc",
    enumerable: true,
  },
});
console.log(obj3); // {c: 'cc'}
console.log(obj3.__proto__); // {p: 'proto'}
const t1 = Object.myAssign({}, obj1, obj2);
console.log(t1); // {a: "aa", b: "bb"}
const t2 = Object.assign({}, obj1, obj3, null, obj2, undefined);
const t3 = Object.myAssign({}, obj1, null, obj2, undefined);
console.log(t3); // {a: "aa", b: "bb"}

//不替换已有的
function merge(target, ...sources) {
  for (let source of sources) {
    for (let key of Object.keys(source)) {
      if (!(key in target)) {
        target[key] = source[key];
      }
    }
  }
  return target;
}

const t4 = Object.assign(
  { x: 1 },
  { x: 2, y: 2 },
  { y: 3, z: 4 }
);
console.log(t4);
//{ x: 2, y: 3, z: 4 }
const t5 = merge(
  { x: 1 },
  { x: 2, y: 2 },
  { y: 3, z: 4 }
);
console.log(t5);
//{ x: 1, y: 2, z: 4 }
```

## 逐行解析

### myAssign 实现

- **`for (let i = 0; i < src.length; i++)`**：遍历所有源对象。
- **`if (src[i] !== null || src[i] !== undefined)`**：过滤掉 `null` 和 `undefined` 的源对象。
- **`for (let key in src[i])`**：`for...in` 会遍历对象及其原型链上的所有可枚举属性。
- **`src[i].hasOwnProperty(key)`**：通过 `hasOwnProperty` 过滤掉原型链上的属性，只复制自身属性。
- **`target[key] = src[i][key]`**：将源对象的属性值浅拷贝到目标对象。

### merge 实现（不覆盖已有属性）

- **`Object.keys(source)`**：只获取对象自身的可枚举属性（不含原型链）。
- **`if (!(key in target))`**：检查目标对象中是否已有该属性，只有不存在时才复制。
- 这种方式适用于需要保留目标对象已有属性的场景。

## 复杂度分析

- **时间复杂度**：O(n × m) — n 为源对象个数，m 为每个源对象的属性个数
- **空间复杂度**：O(1) — 不额外占用空间，直接修改目标对象
