# 统计一组整形数组的最大差值

> 计算数组中最大值与最小值的差值（即极差）。

## Mermaid 流程图

```mermaid
flowchart TD
    A[输入数组 arr] --> B[排序 arr.sort a-b]
    B --> C[取 max = arr[最后一项]]
    B --> D[取 min = arr[第一项]]
    C --> E[返回 max - min]
    D --> E
```

## 源代码

```javascript
const maxDiff = arr => {
    // solution 1
    // arr = arr.flat().sort((a, b) => a - b);
    // return arr[arr.length - 1] - arr[0];

    // solution 2
    let max = 0;
    let min = 0;
    for (let i = 0; i < arr.length; i++) {
        let tempArr = arr.sort((a, b) => a - b);
        min = tempArr[0];
        max = tempArr[tempArr.length - 1] > max ? tempArr[tempArr.length - 1] : max;

    }

    return max - min;
};

let arr2 = [1, 13, 4, 22, 5, 9, 6, 20];
console.log(maxDiff(arr2))
```

## 逐行解析

- **`arr.flat().sort((a, b) => a - b)`**：Solution 1 的思路，先将数组扁平化，然后升序排序，取首尾差值。被注释掉了。
- **`let max = 0; let min = 0`**：初始化最大值和最小值。
- **`for` 循环**：Solution 2 遍历数组，每次都对数组排序，取排序后第一个元素为最小值，最后一个元素为最大值（与当前 max 比较取较大值）。
- **`return max - min`**：返回最大值与最小值的差值。

**注意**：Solution 2 的实现比较低效，每次循环都对整个数组排序，实际上只需遍历一次找出 max 和 min 即可。

## 复杂度分析

| 维度 | 复杂度 | 说明 |
|------|--------|------|
| 时间复杂度 | O(n log n) | 主要来自 `sort()` 排序 |
| 空间复杂度 | O(1) | 原地排序 |
| 优化方案 | O(n) | 一次遍历分别记录 max 和 min，可降低到线性时间 |
