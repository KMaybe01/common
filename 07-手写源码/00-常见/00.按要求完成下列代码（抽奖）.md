# 抽奖系统 —— 白名单优先中奖

> 从参与用户中抽取 2 万个中奖用户，白名单中的活跃用户必定中奖，剩余名额从普通用户中随机选取。

## 源代码

```javascript
// function lottery(whiteList,participant){ your code}
//whiteList：类型字符串数组，意义是表示从其他系统中计算出来的活跃用户，如果这批用户参与抽奖，则必定让他中奖。长度不超过1万。
//participant：类型字符串数组，意义是表示此次活动中真正参与抽奖的用户，长度约10万。
//函数希望从participant返回2万个用户，表示中奖用户，优先选取whiteList上的用户，若不在whiteList上，对participant剩余的随机选取即可。

//方法一
/**
 * @param {*} whiteList 活跃用户
 * @param {*} participant 抽奖用户
 */
function lottery(whiteList, participant){
    const pLen = participant.length,
        wLen = whiteList.length,
        targetNum = 20000
    if (pLen === 0) return [];
    if (pLen < targetNum) return participant;
    let res = [],
        i = 0;
    const map = new Map();
    while (i < wLen && res.length <= targetNum) {
        const pIndex = participant.indexOf(whiteList[i]);
        if (pIndex !== -1) {
            map.set(pIndex, true);
            res.push(whiteList[i]);
        }
        i++;
    }
    while (res.length < targetNum) {
        const index = Math.floor(Math.random() * pLen);
        if (map.has(index)) continue;
        res.push(participant[index]);
        map.set(index.true);
    }
    return res;
}

// 测试
const whiteList = Array.from({
    length: 10000
}, (v, k) => "" + k);
const participant = Array.from({
    length: 100000
}, (v, k) => "" + k);
console.log(lottery(whiteList, participant));

//方法二

function lottery(whiteList, participant) {
    const count = 20000;
    const res = [];
    whiteList.sort();
    participant.sort();

    const pickRandom = () => {
        const idx = parseInt(Math.random() * participant.length);
        res.push(participant.splice(idx, 1)[0]);
    };
    let i1 = 0;
    let i2 = 0;
    while (res.length < count) {
        if (i1 < whiteList.length && i2 < participant.length) {
            if (whiteList[i1] === participant[i2]) {
                let [ele] = participant.splice(i2, 1);
                res.push(ele);
                i1++;
            } else if (whiteList[i1] > participant[i2]) {
                i2++;
            } else {
                i1++;
            }
        } else {
            pickRandom();
        }
    }
    return res;
}
const whiteList2 = Array.from({
    length: 10000
}, (v, k) => "" + k);
const participant2 = Array.from({
    length: 100000
}, (v, k) => "" + k);
console.log(lottery(whiteList2, participant2));
```

## 逐行解析

### 方法一：Map 索引法
- **`const map = new Map()`**：用 Map 记录 participant 中已中奖的用户索引，防止重复抽取。
- **`while (i < wLen && res.length <= targetNum)`**：遍历白名单，通过 `indexOf` 在 participant 中查找白名单用户。找到则加入中奖名单并记录索引。
- **`while (res.length < targetNum)`**：若白名单用户不足 20000 人，从 participant 中随机抽取补足。用 `map.has(index)` 跳过已中奖用户。
- **缺点**：`indexOf` 每次 O(n) 查找，性能较差。`map.set(index.true)` 处有笔误，应为 `map.set(index, true)`。

### 方法二：排序双指针法
- **`whiteList.sort(); participant.sort()`**：先将两个数组排序。
- **`pickRandom()`**：从 participant 剩余数组中随机 `splice` 一个元素加入中奖名单。
- **双指针遍历**：`i1` 指向 whiteList，`i2` 指向 participant。如果当前元素相等，说明是白名单用户，直接加入中奖名单。如果 `whiteList[i1] > participant[i2]`，说明 participant 当前元素不在白名单中，跳过（i2++）。反之 i1++ 跳过白名单中不在 participant 里的用户。
- **优点**：排序后 O(n+m) 线性查找，比方法一的 indexOf 高效。

## 复杂度分析

| 维度 | 方法一（Map索引法） | 方法二（排序双指针法） |
|------|---------------------|----------------------|
| 时间复杂度 | O(w × p) — indexOf 每次 O(p) | O(p log p + w log w + p + w) — 排序为主 |
| 空间复杂度 | O(p) — Map 存储索引 | O(1) — 原地操作 |
| 适用场景 | w 较小时可用 | 大数据量下更优 |
