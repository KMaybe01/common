# 深度优先和广度优先思想实现深拷贝函数

> 分别使用 BFS（广度优先）和 DFS（深度优先）两种遍历策略实现对象的深拷贝，支持处理对象、数组、函数（直接引用）、以及循环引用。

## 源代码

```javascript
// 如果是对象/数组，返回一个空的对象/数组，
// 都不是的话直接返回原对象
// 判断返回的对象和原有对象是否相同就可以知道是否需要继续深拷贝
// 处理其他的数据类型的话就在这加判断
function getEmpty(o) {
    if (Object.prototype.toString.call(o) === '[object Object]') {
        return {};
    }
    if (Object.prototype.toString.call(o) === '[object Array]') {
        return [];
    }
    return o;
}

function deepCopyBFS(origin) {
    let queue = [];
    let map = new Map(); // 记录出现过的对象，用于处理环
    let target = getEmpty(origin);
    if (target !== origin) {
        queue.push([origin, target]);
        map.set(origin, target);
    }
    while (queue.length) {
        let [ori, tar] = queue.shift();
        for (let key in ori) {
            // 处理环状
            if (map.get(ori[key])) {
                tar[key] = map.get(ori[key]);
                continue;
            }
            tar[key] = getEmpty(ori[key]);
            if (tar[key] !== ori[key]) {
                queue.push([ori[key], tar[key]]);
                map.set(ori[key], tar[key]);
            }
        }
    }
    return target;
}

function deepCopyDFS(origin) {
    let stack = [];
    let map = new Map(); // 记录出现过的对象，用于处理环
    let target = getEmpty(origin);
    if (target !== origin) {
        stack.push([origin, target]);
        map.set(origin, target);
    }
    while (stack.length) {
        let [ori, tar] = stack.pop();
        for (let key in ori) {
            // 处理环状
            if (map.get(ori[key])) {
                tar[key] = map.get(ori[key]);
                continue;
            }
            tar[key] = getEmpty(ori[key]);
            if (tar[key] !== ori[key]) {
                stack.push([ori[key], tar[key]]);
                map.set(ori[key], tar[key]);
            }
        }
    }
    return target;
}
// test
[deepCopyBFS, deepCopyDFS].forEach(deepCopy => {
    console.log(deepCopy({
        a: 1
    }));
    console.log(deepCopy([1, 2, {
        a: [3, 4]
    }]))
    console.log(deepCopy(function () {
        return 1;
    }))
    console.log(deepCopy({
        x: function () {
            return "x";
        },
        val: 3,
        arr: [
            1,
            {
                test: 1
            }
        ]
    }))
    let circle = {};
    circle.child = circle;
    console.log(deepCopy(circle));
})
```

## 逐行解析

### getEmpty 辅助函数
- **`Object.prototype.toString.call(o)`**：精确判断数据类型。
- **`[object Object]`**：返回空对象 `{}`。
- **`[object Array]`**：返回空数组 `[]`。
- **其他类型**（函数、基本类型等）：直接返回原值，表示不需要继续深拷贝。

### deepCopyBFS 广度优先深拷贝
- **`queue`**：使用数组模拟队列（先进先出）。
- **`map`**：记录已处理的对象引用，用于解决循环引用问题。
- **初始化**：对根对象 `origin` 调用 `getEmpty` 创建目标空对象。如果 `target !== origin`（说明是对象或数组），将其 `[origin, target]` 对入队并在 map 中记录。
- **`while (queue.length)`**：队列不为空时循环处理。
- **`queue.shift()`**：取出队列头部（广度优先，先入先出）。
- **`for (let key in ori)`**：遍历原对象的所有可枚举属性。
- **`map.get(ori[key])`**：如果该属性值已经被拷贝过（循环引用），直接使用已缓存的对象。
- **`getEmpty(ori[key])`**：否则创建空对象/数组作为副本。如果 `tar[key] !== ori[key]` 说明属性值是对象/数组，需要继续深拷贝，将其入队。

### deepCopyDFS 深度优先深拷贝
- **`stack`**：使用数组模拟栈（后进先出）。
- **`stack.pop()`**：取出栈顶元素（深度优先，后入先出）。
- 其余逻辑与 BFS 版本完全相同，唯一的区别是使用栈（后进先出）替代队列（先进先出）。

### 两种遍历策略的区别
| 特性 | BFS | DFS |
|------|-----|-----|
| 数据结构 | 队列（Queue） | 栈（Stack） |
| 遍历顺序 | 按层级广度优先 | 沿着一个分支深度优先 |
| 对嵌套对象处理 | 先拷贝同层属性，再深入下一层 | 先拷贝一个分支到底，再处理下一个分支 |

## 复杂度分析

| 维度 | 复杂度 | 说明 |
|------|--------|------|
| 时间复杂度 | O(n) | n 为所有需要遍历的节点总数 |
| 空间复杂度 | O(n) | queue/stack 和 map 存储所有节点 |
| 循环引用 | 安全处理 | 使用 Map 记录已处理对象，遇到环直接返回缓存引用 |
