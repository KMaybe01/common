# 基于快速排序分区思想找出最小的 k 个数

> 利用快速排序的 partition 分区算法，找出数组中最小 k 个数字（允许重复，不去重），返回的 k 个数不一定有序。

## Mermaid 流程图

```mermaid
flowchart TD
    A[输入数组 arr, k] --> B[初始化 start=0, end=len-1]
    B --> C[调用 partition 获取 pivot 索引 index]
    C --> D{index === k?}
    D -->|是| E[返回 arr.slice 0, index]
    D -->|否| F{index > k?}
    F -->|是| G[调整 end = index - 1]
    F -->|否| H[调整 start = index + 1]
    G --> C
    H --> C

    subgraph partition 流程
        P1[选最右元素为 pivot] --> P2[i = start - 1]
        P2 --> P3[j 从 start 到 end-1 遍历]
        P3 --> P4{arr[j] <= pivot?}
        P4 -->|是| P5[i++, 交换 arr[i] 和 arr[j]]
        P4 -->|否| P3
        P5 --> P3
        P3 --> P6[交换 arr[i+1] 和 arr[end]]
        P6 --> P7[返回 i+1]
    end
```

## 源代码

```javascript
/**
 * 快排的核心函数，partition分区
 * @param {*} arr  数组
 * @param {*} start  起始下标
 * @param {*} end  结束下标
 */
function partition(arr, start, end) {
    //选最右的数作为pivot
    const pivot = arr[end];
    //指针i初始化为起始下标-1
    let i = start - 1;
    //指针j是从头到倒数第二个数都走一遍
    for (let j = start; j < end; j++) {
        //如果arr[j]比pivot小，就交换arr[i]和arr[j]（把小的移到前面去），且i指针前移
        if (arr[j] <= pivot) {
            i++;
            swap(arr, i, j);
        }
    }
    //最后把pivot换到中间
    swap(arr, i + 1, end);
    //返回pivot的下标
    return i + 1;
}
//交换数组中的两个数
function swap(arr, i, j) {
    const t = arr[i];
    arr[i] = arr[j];
    arr[j] = t;
}
//找出最小的k个数，返回的k个数不一定是有序的
function getLeastNumbers(arr, k) {
    const len = arr.length;
    let start = 0;
    let end = len - 1;
    let index = partition(arr, start, end);
    while (index !== k) {
        if (index > k) {
            end = index - 1;
            index = partition(arr, start, end);
        } else {
            start = index + 1;
            index = partition(arr, start, end);
        }
    }
    return arr.slice(0, index);
}
//测试用例
const arr = [4, 2, 9, 99, 0, 199, 100, 22, 1, 8, 96, 92, 98];
console.log(getLeastNumbers(arr, 3));
```

## 逐行解析

### partition 分区函数
- **`const pivot = arr[end]`**：选中最右侧元素作为基准值。
- **`let i = start - 1`**：`i` 指针指向小于 pivot 的区域的最后一个元素。
- **`for (let j = start; j < end; j++)`**：`j` 指针遍历除 pivot 外的所有元素。
- **`if (arr[j] <= pivot)`**：如果当前元素小于等于 pivot，`i` 前移并交换 `arr[i]` 和 `arr[j]`，将小于等于 pivot 的元素移到左侧。
- **`swap(arr, i + 1, end)`**：遍历结束后，将 pivot 放到 `i+1` 的位置，此时 pivot 左侧全是小于等于它的数。
- **`return i + 1`**：返回 pivot 的索引。

### getLeastNumbers 主函数
- **`while (index !== k)`**：当 pivot 索引不等于 k 时继续循环。
- **`if (index > k)`**：pivot 位置太靠右，说明最小的 k 个数在左半部分，调整 end 为 `index - 1`。
- **`else`**：pivot 位置太靠左，说明最小的 k 个数还有部分在右半部分，调整 start 为 `index + 1`。
- **`return arr.slice(0, index)`**：返回前 k 个元素（无需排序）。

## 复杂度分析

| 维度 | 复杂度 | 说明 |
|------|--------|------|
| 平均时间复杂度 | O(n) | 每次 partition 将问题规模减半：n + n/2 + n/4 + ... = O(2n) |
| 最坏时间复杂度 | O(n²) | 每次选到的 pivot 都是最大或最小值，退化为每次只减少一个元素 |
| 空间复杂度 | O(1) | 原地操作，不使用额外数组 |
| 稳定性 | 不稳定 | 分区过程中会交换元素相对顺序 |
