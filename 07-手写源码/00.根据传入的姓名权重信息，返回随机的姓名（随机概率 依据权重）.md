# 根据传入的姓名权重信息，返回随机的姓名（随机概率 依据权重）

根据权重概率随机返回姓名，权重越高被选中的概率越大。

## 流程图

```mermaid
flowchart TD
    A[输入人员列表<br/>{name, weight}] --> B[reduce 计算权重总和]
    B --> C[标记每个区间的 startW/endW]
    C --> D[生成 0~总和 的随机数 s]
    D --> E[find 查找 s 所属区间]
    E --> F[返回对应的 name]
```

## 代码与解析

```javascript
/**
 * @description: 根据传入的姓名权重信息，返回随机的姓名（随机概率依据权重）
 * @param {Array} personValue
 * @returns {String} personName 姓名
 */
const person = [{
        name: '张三',
        weight: 1
    },
    {
        name: '李四',
        weight: 10
    },
    {
        name: '王五',
        weight: 100
    }
]
var getPersonName = function (personValue) {
    // 标记区间，并且获得weight的总数
    let sum = personValue.reduce((pre, cur) => {
        cur.startW = pre;
        return cur.endW = cur.weight + pre
    }, 0);
    let s = Math.random() * sum; // 获得一个 0 - 111 的随机数
    // 判断随机数的所属区间
    let person = personValue.find(item => item.startW < s && s <= item.endW);
    return person.name;
}

function getResult(count) {
    const res = {}
    for (let i = 0; i < count; i++) {
        const name = getPersonName(person)
        res[name] = res[name] ? res[name] + 1 : 1
    }
    console.log(res)
}
getResult(10000) //{ '王五': 9003, '李四': 913, '张三': 84 } 随机
```

- **区间标记**：`reduce` 遍历人员列表，每个元素记录 `startW`（区间起点）和 `endW`（区间终点）
  - 张三：`[0, 1)`
  - 李四：`[1, 11)`
  - 王五：`[11, 111)`
- **权重总和**：`sum = 1 + 10 + 100 = 111`
- **随机命中**：`Math.random() * sum` 生成 `[0, 111)` 随机数
- **区间查找**：`find` 查找随机数落在哪个区间，返回对应姓名
- **结果验证**：`getResult(10000)` 运行 10000 次，王五（权重 100）约 90%，李四（权重 10）约 9%，张三（权重 1）约 1%

## 复杂度分析

| 操作 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 区间标记（一次） | O(n) | O(1) — 原地标记 |
| 单次随机抽取 | O(n) — find 遍历 | O(1) |
| 批量测试 N 次 | O(N × n) | O(k) — k 为不同姓名数 |
