# 实现图片的加载

## 简介

使用 Promise 封装图片加载函数，支持链式调用依次加载多张图片并追加到页面中。

## 执行流程

```mermaid
flowchart TD
    A[开始] --> B[调用 createImg(url)]
    B --> C{url 存在?}
    C -->|否| D[reject 抛出错误]
    C -->|是| E[创建 img 元素]
    E --> F[设置 src 属性]
    F --> G[resolve 返回 img 元素]
    G --> H[then 中 appendChild]
    H --> I{还有下一张图片?}
    I -->|是| J[返回下一个 createImg]
    J --> H
    I -->|否| K[结束]
```

## 代码实现

```javascript
function createImg(url) {
    return new Promise((resolve, reject) => {
        if (url) {
            let ImgEle = document.createElement("img");
            ImgEle.src = url;
            resolve(ImgEle);
        } else {
            reject("url is not right");
        }
    });
}


createImg(
    "https://ss3.bdstatic.com/70cFv8Sh_Q1YnxGkpoWK1HF6hhy/it/u=3612597965,1770541226&fm=26&gp=0.jpg"
).then((value) => {
    document.body.appendChild(value);
    return createImg("2.jpg");
}).then(function (value) {
    document.body.appendChild(value);
    return createImg("3.jpg");
});
```

## 逐行解析

1. **第 6 行**: 定义 `createImg` 函数，返回一个 Promise。
2. **第 7 行**: 创建 Promise，接收 `resolve` 和 `reject` 两个参数。
3. **第 8-11 行**: 如果 `url` 有效，创建 `img` 元素并设置 `src`，然后 `resolve` 返回该元素。
4. **第 12-14 行**: 如果 `url` 无效，调用 `reject` 返回错误信息。
5. **第 19-27 行**: 链式调用 `.then()`，每次将加载的图片追加到 `body`，并返回下一个 `createImg` 调用，实现顺序加载。

## 复杂度分析

- **时间复杂度**: O(n)，n 为图片数量，每张图片顺序加载。
- **空间复杂度**: O(1)，每次仅保留当前处理的图片引用。
