# 只执行三次的函数实现（闭包控制）

## 简介

使用闭包控制函数执行次数，通过 `times` 变量计数，确保函数最多执行三次。

## 执行流程

```mermaid
flowchart TD
    A[调用 setFn(fn)] --> B[初始化 times = 0]
    B --> C[返回闭包函数]
    C --> D[再次调用时]
    D --> E{times++ < 3?}
    E -->|是| F[执行 fn(times)]
    E -->|否| G[不执行]
    F --> H[结束]
    G --> H
```

## 代码实现

```javascript
function getName(value) {
    console.log(value);
}

function setFn(fn) {
    let times = 0;
    return () => {
        if (times++ < 3) {
            fn(times);
        }
    };
}

const newFn = setFn(getName);
newFn(); // 1
newFn(); // 2
newFn(); // 3
newFn(); // 无效
newFn(); // 无效
```

## 逐行解析

1. **第 1-3 行**: 定义 `getName` 函数，接收参数并打印。
2. **第 5 行**: 定义 `setFn` 高阶函数，接收一个函数 `fn`。
3. **第 6 行**: 初始化 `times` 为 0，闭包持有该变量。
4. **第 7-11 行**: 返回一个箭头函数，每次调用时判断 `times++ < 3`，满足则执行 `fn(times)`。
5. **第 14 行**: 调用 `setFn(getName)` 返回受控制的 `newFn` 函数。
6. **第 15-19 行**: 前三次调用正常执行输出 1、2、3，后两次调用因 `times` 已到 3 不再执行。

## 复杂度分析

- **时间复杂度**: O(1)，每次调用仅执行常数量操作。
- **空间复杂度**: O(1)，闭包仅保留 `times` 一个数字变量。
