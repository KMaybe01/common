# 手写数组去重的方法

## 简介

数组去重是前端面试中的高频题目。本文总结了 6 种常用去重实现方式：Set、对象属性法、includes/indexOf 法、排序法、splice 双重循环法、filter + indexOf 法。它们各自有不同的适用场景和性能特征。

## 执行流程

```mermaid

    subgraph Set
        A1[new Set(arr)] --> A2[Array.from 转数组]
    end
    subgraph 对象属性
        B1[遍历数组] --> B2{obj[cur] 已存在?}
        B2 -->|是| B3[删除当前元素]

    end

        C1[遍历数组] --> C2{newAry.includes(cur)?}
        C2 -->|否| C3[push 到新数组]
    end
    subgraph filter
        D1[arr.filter] --> D2{indexOf(item) === index}
        D2 -->|是| D3[保留]
    end
```

## 代码实现

```javascript
let arr = [12, 1, 12, 3, 1, 88, 66, 9, 66];

// 1. Set
function unique(ary) {
    let s = new Set(ary);
    return Array.from(s)
}
unique(arr);

// 2. 展开运算符 + Set
let a = [...new Set(arr)];
console.log(a);

// 3. 对象属性名不能重复
function unique(ary) {
    let obj = {};
    for (let i = 0; i < ary.length; i++) {
        let cur = ary[i];
        if (obj[cur]) {
            ary[i] = ary[ary.length - 1];
            ary.length--;
            i--;
            continue;
        }
        obj[cur] = cur;
    }
}
unique(arr);

// 4. includes / indexOf
function unique(ary) {
    let newAry = [];
    let len = ary.length;
    for (let i = 0; i < len; i++) {
        let cur = ary[i];
        if (!newAry.includes(cur)) {
            newAry.push(cur);
        }
    }
    return newAry;
}
console.log(unique(arr));

// 5. sort + splice
function unique(ary) {
    let a = ary.sort((a, b) => a - b);
    for (let i = 0; i < a.length; i++) {
        if (a[i] === a[i + 1]) {
            a.splice(i + 1, 1);
            i--;
        }
    }
    return a;
}
unique(arr)

// 6. splice 双重循环
function unique(ary) {
    for (let i = 0; i < ary.length; i++) {
        for (j = i + 1; j < ary.length; j++) {
            if (ary[i] === ary[j]) {
                ary.splice(j, 1);
                j--;
            }
        }
    }
    return ary;
}
unique(arr);

// 7. filter + indexOf
function unique(ary) {
     return ary.filter((item,index)=> ary.indexOf(item) === index)
}
console.log(unique(arr));
```

## 逐行解析

### 方法 1：Set
- `new Set(ary)` 创建一个 Set 对象，Set 中所有元素都是唯一的。
- `Array.from(s)` 将 Set 数据结构转回数组。
- 最简单、最推荐的方式，时间复杂度低。

### 方法 2：展开运算符 + Set
- `[...new Set(arr)]` 利用展开运算符直接将 Set 转为数组，一行代码解决。

### 方法 3：对象属性法
- 利用对象的属性名不能重复的特性。
- 遍历数组，以元素值作为 `obj` 的属性名。如果已存在，通过将当前元素替换为数组最后一个元素 + `length--` 的方式删除（避免 splice 导致的数组塌陷）。

### 方法 4：includes / indexOf
- 创建一个新数组 `newAry`，遍历原数组，如果新数组中不包含当前元素则 push 进去。
- `includes` 在底层也是遍历比较，性能中等。

### 方法 5：sort + splice
- 先排序，让相同元素相邻。
- 遍历排序后的数组，如果当前元素与下一个相同，则用 splice 移除下一个元素。

### 方法 6：splice 双重循环
- 外层循环遍历每个元素，内层循环从 i+1 开始向后查找相同元素，找到后用 splice 删除。
- 删除后需要 `j--` 防止跳过下一个元素。

### 方法 7：filter + indexOf
- `arr.filter((item, index) => arr.indexOf(item) === index)`：`indexOf` 返回元素第一次出现的位置，只有当前位置等于第一次出现位置时才保留，从而实现去重。

## 复杂度分析

| 方法 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| Set | O(n) | O(n) |
| 展开+Set | O(n) | O(n) |
| 对象属性法 | O(n) | O(n) |
| includes | O(n²) | O(n) |
| sort+splice | O(n log n) | O(1) |
| splice双重循环 | O(n²) | O(1) |
| filter+indexOf | O(n²) | O(n) |
