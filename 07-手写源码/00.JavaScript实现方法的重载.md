# JavaScript 实现方法的重载

## 简介

JavaScript 本身不支持传统语言（如 Java、C++）中的方法重载。但可以通过闭包和参数个数判断来模拟重载：利用 `arguments.length` 与函数形参个数 `fn.length` 进行比较，根据参数个数的不同调用不同的函数实现。

## 执行流程

```mermaid
flowchart TD
    A[调用 methods.add(arg1, arg2, ...)] --> B[从最晚注册的函数开始检查]
    B --> C{参数个数 fn.length === arguments.length?}
    C -->|是| D[执行当前函数]
    C -->|否| E{存在 old 函数?}
    E -->|是| F[调用 old 继续检查]
    E -->|否| G[无匹配函数]
```

## 代码实现

```javascript
function addMethod(object, name, fnt) {
    var old = object[name];  // 保存前一个值，以便后续调用
    object[name] = function(){  // 向object[name]赋值一个代理函数
      // 判断fnt期望接收的参数与传入参数个数是否一致
      if (fnt.length === arguments.length)
        // 若是，则调用fnt
        return fnt.apply(this, arguments)
      else if (typeof old === 'function')  // 若否，则判断old的值是否为函数
        // 若是，则调用old
        return old.apply(this, arguments);
    };
  }
  //模拟重载add
  var methods = {};
  //添加方法，顺序无关
  addMethod(methods, 'add', function(){return 0});
  addMethod(methods, 'add', function(a,b){return a + b});
  addMethod(methods, 'add', function(a,b,c){return a + b + c});
  //执行
  console.log(methods.add()); //0
  console.log(methods.add(10,20)); //30
  console.log(methods.add(10,20,30)); //60
```

## 逐行解析

- **`addMethod(object, name, fnt)`**：重载注册函数，接收三个参数：目标对象、方法名、实现函数 `fnt`。
- **`var old = object[name]`**：闭包保存当前 `object[name]` 的旧值（可能是前一次注册的函数或者 `undefined`）。
- **`object[name] = function() {...}`**：向 `object[name]` 赋值为一个新的代理函数。
- **`fnt.length === arguments.length`**：检查当前注册函数的期望参数个数与实际传入个数是否相等。`fn.length` 是函数的形参个数。
- **`fnt.apply(this, arguments)`**：如果参数个数匹配，则调用当前函数。
- **`typeof old === 'function'`**：参数不匹配时，检查之前保存的 `old` 是否为函数，如果是则调用 `old`（回到上一层的代理函数继续匹配）。
- **注册顺序**：后注册的函数先被匹配，因为新函数覆盖了旧函数，形成一个闭包链。

## 复杂度分析

- **时间复杂度**：O(n) — 最多需要遍历 n 个注册函数才能找到匹配项
- **空间复杂度**：O(n) — 每个注册的函数都通过闭包持有上一个函数的引用，形成闭包链
