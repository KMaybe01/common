# 判断子序列

## 简介

判断数组 `b` 是否为数组 `a` 的子序列。子序列要求元素以原始相对顺序出现于 `a` 中，不要求连续。

## 执行流程

```mermaid
flowchart TD
    A[开始] --> B[初始化 bi=0 ai=0]
    B --> C{bi < b.length?}
    C -->|否| D[返回 true]
    C -->|是| E{a[ai] === b[bi]?}
    E -->|是| F[bi++ 匹配成功]
    E -->|否| G[ai++ 继续搜索]
    F --> H{ai > a.length?}
    G --> H
    H -->|是| I[返回 false]
    H -->|否| C
    D --> J[结束]
    I --> J
```

## 代码实现

```javascript
const isSubsequence = (b, a) => {
    let bi = 0,
        ai = 0;
    while (bi < b.length) {
        if (a[ai] === b[bi]) {
            bi++;
        } else {
            ai++;
        }

        if (ai > a.length) {
            return false;
        }
    }
    return true;
};

const t1 = isSubsequence([1, 5, 11], [1, 3, 5, 7, 11]);
const t2 = isSubsequence([1, 11],  [1, 3, 5, 7, 11]);
const t3 = isSubsequence([1, 11, 5],  [1, 3, 5, 7, 11]);
console.log(t1) // true
console.log(t2) // true
console.log(t3) // false
```

## 逐行解析

1. **第 3-4 行**: 定义 `isSubsequence`，使用双指针 `bi`（子序列索引）和 `ai`（主数组索引）。
2. **第 6-16 行**: `while` 循环遍历，若 `a[ai] === b[bi]` 则匹配成功同时移动两指针，否则仅移动 `ai`。若 `ai` 超出主数组长度说明无法匹配全部元素，返回 `false`。
3. **第 20-22 行**: 测试 — `[1,5,11]` 和 `[1,11]` 是子序列，`[1,11,5]` 因为顺序不符不是子序列。

## 复杂度分析

- **时间复杂度**: O(n + m)，n 为子序列长度，m 为主数组长度，双指针各遍历一次。
- **空间复杂度**: O(1)，仅使用两个指针变量。
