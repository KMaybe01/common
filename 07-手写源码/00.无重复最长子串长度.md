# 无重复最长子串长度

## 简介

给定一个字符串，找出其中不含有重复字符的最长子串的长度。本文提供两种实现方式：Map 滑动窗口和数组滑动窗口。

## 流程图

```mermaid
flowchart TD
    A[开始] --> B[初始化 map/i=-1/res=0]
    B --> C[遍历字符串 j=0 to n-1]
    C --> D{map.has s[j]?}
    D -->|是| E[i = max i, map.get s[j]]
    D -->|否| F[res = max res, j-i]
    E --> F
    F --> G[map.set s[j], j]
    G --> C
    C -->|遍历结束| H[返回 res]
    H --> I[结束]
```

```mermaid
flowchart TD
    A[开始] --> B[初始化 arr=[] / max=0]
    B --> C[遍历字符串 i=0 to len-1]
    C --> D[arr.indexOf s[i]]
    D --> E{index !== -1?}
    E -->|是| F[arr.splice 0, index+1]
    E -->|否| G[arr.push s[i]]
    F --> G
    G --> H[max = max arr.length max]
    H --> C
    C -->|结束| I[返回 max]
```

## 代码实现

```javascript
// 方法一：Map 滑动窗口
var lengthOfLongestSubstring = function (s) {
    let map = new Map();
    let i = -1
    let res = 0
    let n = s.length
    for (let j = 0; j < n; j++) {
        if (map.has(s[j])) {
            i = Math.max(i, map.get(s[j]))
        }
        res = Math.max(res, j - i)
        map.set(s[j], j)
    }
    return res
};

// 方法二：数组滑动窗口
var lengthOfLongestSubstring = function (s) {
    let arr = [];
    let max = 0;
    for (let i = 0; i < s.length; i++) {
        let index = arr.indexOf(s[i])
        if (index !== -1) {
            arr.splice(0, index + 1);
        }
        arr.push(s.charAt(i));
        max = Math.max(arr.length, max);
    }
    return max;
};
```

## 逐行解析

### 方法一（Map 滑动窗口）
- **第3行**：定义函数，接收字符串 `s`
- **第4行**：创建 Map 用于存储字符及其索引位置
- **第5行**：`i` 表示左边界，初始为 -1
- **第6行**：`res` 记录最大长度
- **第7行**：`n` 为字符串长度
- **第8行**：`j` 作为右边界遍历字符串
- **第9-11行**：如果当前字符已在 Map 中，将左边界 `i` 移动到上一次出现位置的右侧（取较大值防止左移）
- **第12行**：计算当前窗口长度 `j - i`，更新最大值
- **第13行**：更新当前字符的最新索引
- **第15行**：返回最大长度

### 方法二（数组滑动窗口）
- **第21行**：定义函数
- **第22行**：`arr` 作为滑动窗口数组
- **第24行**：查找当前字符在窗口中的位置
- **第25-27行**：如果字符已存在，删除它及其之前的所有元素
- **第28行**：将当前字符加入窗口
- **第29行**：更新最大长度
- **第31行**：返回结果

## 复杂度分析

| 方法 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| Map 滑动窗口 | O(n) | O(min(m, n)) |
| 数组滑动窗口 | O(n²) | O(min(m, n)) |

其中 n 为字符串长度，m 为字符集大小。
