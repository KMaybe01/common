# 生成指定范围的随机数

生成 `[min, max]` 闭区间内的随机整数。

## 流程图

```mermaid
flowchart TD
    A[输入 min, max] --> B[计算范围 range = max - min + 1]
    B --> C[生成 0~1 随机数<br/>Math.random]
    C --> D[乘以 range<br/>Math.floor 取整]
    D --> E[加上 min 偏移]
    E --> F[返回结果]
```

## 代码与解析

```javascript
export const randomNum = (min, max) =>
  Math.floor(Math.random() * (max - min + 1)) + min;
```

- `Math.random()` 生成 `[0, 1)` 浮点数
- `(max - min + 1)` 是闭区间的范围长度
- `Math.floor(Math.random() * range)` 生成 `[0, range-1]` 的整数
- 加上 `min` 偏移到目标区间 `[min, max]`
- 例如 `randomNum(5, 10)` 能生成 5、6、7、8、9、10 中的任意整数
