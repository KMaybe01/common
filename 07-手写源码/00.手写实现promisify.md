# 手写实现 promisify

## 简介

`promisify` 将一个回调风格的函数（如 Node.js 的 `fs.readFile`）转换为返回 Promise 的函数。支持回调参数在前或在后的两种模式。

## 流程图

```mermaid
flowchart TD
    A[promisify fn, reverse] --> B{fn 是否为函数?}
    B -->|否| C[throw TypeError]
    B -->|是| D[返回新函数]
    D --> E[新函数接收 ...args]
    E --> F[返回 new Promise]
    F --> G{reverse === true?}
    G -->|是| H[fn.call null callback, ...args]
    G -->|否| I[fn.call null ...args, callback]
    H --> J[try/catch 包裹]
    I --> J
    J -->|callback 执行| K{args[0] 是 Error?}
    K -->|是| L[reject error]
    K -->|否| M[resolve args]
    J -->|catch err| N[reject err]
```

## 代码实现

```javascript
function promisify(fn, reverse) {
    if ({}.toString.call(fn) !== '[object Function]') {
        throw new TypeError('Only normal function can be promisified');
    }
    return function (...args) {
        return new Promise((resolve, reject) => {
            const callback = function (...args) {
                if ({}.toString.call(args[0]) === '[object Error]') {
                    return reject(args[0]);
                }
                resolve(args);
            };
            try {
                if (reverse === true) {
                    fn.apply(null, [callback, ...args]);
                } else {
                    fn.apply(null, [...args, callback]);
                }
            } catch (err) {
                reject(err);
            }
        });
    }
}

// 示例
function foo(str1, str2, callback) {
    setTimeout(() => {
        console.log('wait a minute')
        callback(str1, str2)
    }, 1000)
}

let agent = promisify(foo)
agent('hello', 'world').then(res => {
    console.log(res)
})
```

## 逐行解析

- **第6行**：类型检查，确保 `fn` 是函数
- **第7-9行**：不是函数则抛出类型错误
- **第10行**：返回一个新的函数，接收调用参数
- **第11行**：返回一个 Promise
- **第12-17行**：定义 `callback` 函数，检查第一个参数是否为 Error 对象（Node.js 回调约定），是则 reject，否则 resolve
- **第18-24行**：根据 `reverse` 参数决定回调的位置。`reverse=true` 时回调放在参数最前面（如 `setTimeout`），否则放在最后（Node.js 标准风格）
- **第19-23行**：try/catch 捕获同步异常
- **第32-38行**：示例函数 `foo`，模拟异步操作，调用回调
- **第42-47行**：使用 `promisify` 转换后可以用 `.then` 风格调用

## 复杂度分析

- **时间复杂度**：O(1)（创建包装函数的开销）
- **空间复杂度**：O(1)
