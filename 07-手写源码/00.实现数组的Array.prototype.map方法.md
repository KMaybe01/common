# 实现数组的 Array.prototype.map 方法

## 简介

手动实现数组的 `map` 方法，对数组的每个元素执行回调函数，返回一个新数组，不改变原数组。

## 执行流程

```mermaid
flowchart TD
    A[开始] --> B[检查参数合法性]
    B --> C{arr是数组且\nmapCallback是函数?}
    C -->|否| D[返回空数组 []]
    C -->|是| E[初始化空数组 result]
    E --> F[遍历数组每个元素]
    F --> G[调用 mapCallback(arr[i], i, arr)]
    G --> H[将返回值 push 到 result]
    H --> I{遍历完成?}
    I -->|否| F
    I -->|是| J[返回 result]
    J --> K[结束]
```

## 代码实现

```javascript
function map(arr, mapCallback) {
    // 首先，检查传递的参数是否正确。
    if (!Array.isArray(arr) || !arr.length || typeof mapCallback !== 'function') {
        return [];
    } else {
        let result = [];
        // 每次调用此函数时，我们都会创建一个 result 数组
        // 因为我们不想改变原始数组。
        for (let i = 0, len = arr.length; i < len; i++) {
            result.push(mapCallback(arr[i], i, arr));
            // 将 mapCallback 返回的结果 push 到 result 数组中
        }
        return result;
    }
}
```

## 逐行解析

1. **第 9 行**: 定义 `map` 函数，接收两个参数：待处理数组 `arr` 和回调函数 `mapCallback`。
2. **第 11 行**: 参数校验 — 检查 `arr` 是否为数组、是否为空、`mapCallback` 是否为函数，任一不满足则返回空数组。
3. **第 14 行**: 初始化空数组 `result` 用于存储处理后的结果。
4. **第 17 行**: 使用 `for` 循环遍历原始数组，注意缓存 `len = arr.length` 避免每次循环重新读取长度。
5. **第 18 行**: 调用 `mapCallback(arr[i], i, arr)`，传入当前元素、索引和原数组，将返回值推入 `result`。
6. **第 21 行**: 返回新数组 `result`，原数组保持不变。

## 复杂度分析

- **时间复杂度**: O(n)，n 为数组长度，每个元素执行一次回调。
- **空间复杂度**: O(n)，需要创建一个与原数组等长的新数组存储结果。
