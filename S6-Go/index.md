4. 面试前：只看"💡 记忆关键词"和"📝 一句话总结"快速回

## 📚 附录：实战题库索引（50 道精选编程题）

|------|------|------|----------|

| ⭐ | 有效的括号 | 栈 | `[]rune` 模拟 |

| ⭐⭐ | LRU 缓存 | Map+双向链表 | `container/list` |
| ⭐⭐⭐ | 实现并发安全的限流器 | 令牌桶 | `time.Ticker` + `sync.Mutex` |
| ⭐⭐⭐ | 解析 JSON 并提取指定字段 | 递归/反射 | `encoding/json` + `interface{}` |
| ⭐⭐⭐⭐ | 实现简易 HTTP 路由 | 前缀树 | Radix Tree |
| ⭐⭐⭐⭐ | 分布式 ID 生成器 | Snowflake | 时间戳+机器ID+序列号 |
