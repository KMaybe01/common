## 🚀 项目实践

### 项目一：5G核心网测试用例管理系统
**技术栈**：React 19 + UmiJS + SSE + Kubernetes + Go

> **背景**：5G 核心网网元（SMF / UPF）测试流程复杂，涉及 **7 种网元配置**、**30+ API 接口**、**12 个业务子组件**。传统 UI 编码方式无法支撑测试用例频繁变动，每次测试场景变更需开发介入改代码发版，严重制约测试效率。

- **动态表单 DSL 引擎** —— 设计**后端驱动的 JSON Schema 渲染引擎**，支持**无限层级嵌套**（tabs → card → form → leaf fields）与 **6 种字段类型**，实现**非前端人员零代码配置**测试场景，同类需求开发人效提升 **80%**
- **树形数据操作引擎** —— 自研**递归 CRUD 操作库**，实现事件树的 **Key 唯一性保证算法**、操作权限 Map 生成与递归校验，支撑复杂测试用例的可视化编辑
- **React 18 并发特性实战** —— 在**可编辑树表格（628 行）**中应用 **useDeferredValue + startTransition**，**500ms 防抖校验** + DOM 错误高亮，编辑性能提升 **40%**
- **智能诊断流** —— 利用 **ReadableStream 构建实时日志流式传输**，AbortController 支持请求取消，结合正则算法实现异常状态**秒级高亮**，问题定位效率提升 **50%**
- **云原生运维** —— 实现 **Kubernetes YAML 在线编辑与下发**，将前端作为云原生网元的管理入口，支撑日均 **200+** 自动化测试任务

### 项目二：AeMS— 企业级综合网络管理系统
**技术栈**：Angular 20 + RxJS + WebSocket + OpenLayers + ECharts

> **背景**：面向 **数十万台 Small Cell 网元设备** 的综合智能网管中枢，覆盖设备监控、智能告警、日志分析、安全审计全链路。系统包含 **35+ 页面**、**100+ 组件**、**30+ API 服务**，需处理 **1000+ QPS** 告警并发与 **十万级设备** GIS 拓扑渲染。

- **十万级设备地图** —— 设计 **BBOX 视口裁剪 + 聚合显示 + dataCache 全量缓存 + moveend 懒刷新**四重优化策略，将**十万设备渲染降至百级点位**，自定义 EPSG:900913 投影支持坐标双向转换
- **声明式 API 服务层** —— 基于 **TypeScript 装饰器**实现 @GET/@POST/@PATH/@BODY 声明式 HTTP 抽象，**Proxy/Reflect 拦截器透明代理**，消除 **200+** 个接口方法的样板代码
- **LRU 路由缓存** —— 实现 **RouteReuseStrategy 四个方法**，LRU 淘汰（上限 6）+ 滚动位置恢复 + 跨模块级清理 + 手动 destroy，页面切换性能提升 **60%**
- **位编码权限体系** —— 设计**树形常量树自动生成 369 个权限码映射**，三层联动控制（菜单递归过滤 → 路由 Guard 拦截 → 按钮 ACL），实现 **100+ 权限点的精细化管控**
- **精确 Loading 管理** —— **Signal 驱动的请求级粒度追踪**，loadingInterceptor 按 `METHOD /path` 标记，ButtonGroupComponent 自动关联，消除全局 Loading 闪烁

### 项目三：网元运维与数据管理系统
**技术栈**：Angular 20 + PrimeNG + Tailwind CSS + Go + Prometheus + Grafana

> **背景**：5G 核心网元（SMF / UPF）的运维与数据管理系统，覆盖网元全生命周期：注册接入、状态监控、配置下发、日志采集解密、备份恢复、审计溯源。前端 **40+ 组件**、**9 个 API 服务**、**4 层拦截器链**，需满足电信级安全合规要求（**HKDF + AES-256-GCM 加密存储**）。

- **Web Worker 并行解密** —— 设计 **Worker Pool 分治 + 有序合并 + 流式输出三阶段策略**，根据 CPU 核心数动态创建 **2-3 个 Worker 并行 RSA 解密**，首段小分区实现**快速首屏渲染**，大文件前 10% 流式输出避免 UI 冻结
- **自研表单系统** —— 采用**注册表 + 工厂模式**设计声明式表单框架，FormBase 抽象类 + NgComponentOutlet 动态渲染，支持 **9 种控件类型**与 **5 个跨字段验证器**，新增控件只需继承并注册
- **安全架构** —— 实现 **JWT + JTI 单点登录**（新登录自动踢出旧会话）、**HKDF + AES-256-GCM 审计日志加密**（每行独立加密 + tar.gz 轮转）、bcrypt 密码哈希 + 时序攻击防护 + 自动锁定**三层防暴力破解**
- **流式数据处理** —— 封装 **Fetch API 的 Observable StreamingService**，ReadableStream 逐块读取 + TextDecoder stream 模式处理多字节字符，AbortController 支持请求取消，与 Angular 异步管道无缝集成
- **可观测性建设** —— 集成 **Prometheus 暴露 9 个前端指标**，Zoneless + Signals **零开销变更检测**，ESLint 强制 500 行文件限制 + 圈复杂度 20，TypeScript strict 模式全量开启
