1. 课程简介 ​

AI Agent（人工智能智能体）是能够自主感知环境、做出决策、执行动作的软件实体。区别于传统程序，Agent 具备：

  • 感知能力：接收用户指令、环境反馈
  • 规划能力：分解复杂任务、制定执行步骤
  • 工具调用：连接外部 API、数据库、浏览器
  • 记忆能力：存储对话历史、学习经验

本课程以 "编程智能体" 为实战目标，从零构建一个能够：

  • 理解自然语言编程需求
  • 操作终端执行命令
  • 控制浏览器进行测试
  • 读写数据库
  • 在沙盒中自动编码部署
  • 通过 MCP 协议扩展无限工具

的商业级 AI Agent 系统。

6. MCP 协议 ​

6.1 MCP 原理 ​

MCP（Model Context Protocol）是 Anthropic 提出的开放协议，用于让 AI 模型安全地访问外部工具和数据源。

6.2 MCP 传输层 ​

传输方式描述适用场景
stdio通过标准输入/输出通信本地工具，子进程方式运行
SSEServer-Sent Events over HTTP远程工具，网络服务

6.3 用 TypeScript 编写 MCP 服务器（stdio） ​

typescript
// mcp-server.ts
import { Server } from "@modelcontextprotocol/sdk/server/index.js";
import { StdioServerTransport } from "@modelcontextprotocol/sdk/server/stdio.js";
import {
  ListToolsRequestSchema,
  CallToolRequestSchema,
} from "@modelcontextprotocol/sdk/types.js";

const server = new Server(
  { name: "my-tools", version: "1.0.0" },
  { capabilities: { tools: {} } }
);

server.setRequestHandler(ListToolsRequestSchema, async () => ({
  tools: [
    {
      name: "get_weather",
      description: "获取城市天气",
      inputSchema: {
        type: "object",
        properties: {
          city: { type: "string", description: "城市名" },
        },
        required: ["city"],
      },
    },
    {
      name: "execute_shell",
      description: "执行 shell 命令",
      inputSchema: {
        type: "object",
        properties: {
          command: { type: "string", description: "命令" },
        },
        required: ["command"],
      },
    },
  ],
}));

server.setRequestHandler(CallToolRequestSchema, async (request) => {
  const { name, arguments: args } = request.params;

  switch (name) {
    case "get_weather":
      return {
        content: [{ type: "text", text: `${args.city}: 25°C, 晴` }],
      };

    case "execute_shell": {
      const { execSync } = await import("child_process");
      const output = execSync(args.command, { encoding: "utf-8" });
      return {
        content: [{ type: "text", text: output }],
      };
    }

    default:
      throw new Error(`未知工具: ${name}`);
  }
});

const transport = new StdioServerTransport();
await server.connect(transport);

class HTMLOptimizer {
  optimize(html: string): string {
    return html
      .replace(/<script[\s\S]*?<\/script>/gi, "")
      .replace(/<style[\s\S]*?<\/style>/gi, "")
      .replace(/<!--[\s\S]*?-->/g, "")
      .replace(/<svg[\s\S]*?<\/svg>/gi, "")
      .replace(/<[^>]+style="[^"]*display:\s*none[^"]*"[^>]*>/gi, "")
      .replace(/>\s+</g, "><")
      .replace(/\s{2,}/g, " ")
      .trim();
  }

  extractText(html: string): string {
    return this.optimize(html)
      .replace(/<br\s*\/?>/gi, "\n")
      .replace(/<[^>]+>/g, "")
      .replace(/&nbsp;/g, " ")
      .replace(/&amp;/g, "&")
      .replace(/&lt;/g, "<")
      .replace(/&gt;/g, ">")
      .replace(/\n{3,}/g, "\n\n")
      .trim();
  }
}

      stream.on("error", reject);

  .addEdge("analyze", "plan")
  .addEdge("plan", "execute")
  .addEdge("execute", "__end__");
